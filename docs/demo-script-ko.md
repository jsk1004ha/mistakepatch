# 60-Second Demo Script (Korean)

1. 0~10초
- "기존 카메라 풀이 앱은 정답 중심이고, MistakePatch는 내 풀이 감점 피드백 중심입니다."
- 풀이 이미지 업로드

2. 10~25초
- "루브릭 10점 기준으로 총점과 감점 사유를 구조화해서 보여줍니다."
- 점수 카드 + 감점 카드 표시

3. 25~40초
- "감점 카드를 누르면 틀린 위치를 하이라이트합니다."
- 카드 클릭 후 이미지 탭으로 동그라미 저장

4. 40~52초
- "정답 전체 대신 최소 수정 패치만 제안합니다."
- 패치 탭에서 변경 포인트 확인

5. 52~60초
- "히스토리와 Top3 태그로 반복 실수를 추적합니다."
- 히스토리 패널 + Top3 태그 확인
